# taoyao-todo
桃夭todo是桃夭为了自己顺手而开发的功能简单的todo list小软件，目前已经被用于桃夭的个人生活。
该小软件由html和js文件组成，结构简单。前端和后端全部跑在轻量级阿里云服务器上，目前只支持简单的苹果桌面快捷方式搭载。
1.0版本于2026年3月31日发布，因为完全根据桃夭个人的手癖设计，目前支持以下功能：

1.综合任务表：支持编写每周综合任务，按照DDL和紧迫度进行排列（因为桃夭看着顺眼）
2.每日任务表/人工安排：支持每日任务安排，可以自由安排完成任务时段，编写紧迫度以供评估每日日程安排（依旧桃夭看着顺眼）
3.每日任务表/智能安排：支持任务输入，填写可供安排的工作时间后打包为Json格式数据发送给AI进行智能安排填写（因为桃夭很懒）

由于API收费问题暂不开放，桃夭自己用。
